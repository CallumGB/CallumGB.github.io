Riveal 11.1

by Ron Hayter
mystriveal@mac.com

Riveal is a free application that extracts media (sounds, movies, and pictures) from the data files of the Myst series of games and from other games produced by Cyan Worlds and Presto Studios. It also extracts media from the unrelated but excellent games Alida, BioShock, BioShock 2, BioShock Infinite, Borderlands, Borderlands 2, and Infinity Blade I, II, and II.

There are two versions of Riveal, one for OS X and the other for Windows.

Download Riveal for OS X.

Download Riveal for Windows.

Being a Java application, Riveal requires a Java SE 6 (or later) runtime. Mac OS X 10.5 (Leopard) and 10.6 (Snow Leopard) have Java 6 built in; OS X Lion (10.7), OS X Mountain Lion (10.8), OS X Mavericks (10.9), and OS X Yosemite (10.10) will offer to install Java, if needed. For Windows, you may need to download and install Java, which you can get from http://www.java.com/en/download/manual.jsp.

Running Riveal

OS X version:

  • Double-click Riveal.
  • It will ask you to point to a game's data files.
    • For many games, choose a folder (or a folder of folders) containing the game's data files.
    • For Myst IV: Revelation, Myst V: End of Ages, Myst Online: Uru Live, BioShock 2, BioShock Infinite, Borderlands, or Borderlands 2, choose the game's application file. Some games will be in the "/Applications" folder. Steam games are in the "/Users/yourID/Library/Application Support/Steam/SteamApps/common" folder.
    • For BioShock, be sure to choose the folder containing both the BioShock application and the "BioShockData.dmg" disk image.
    • For an iOS game, you must choose the app file (".ipa") on your Mac. Look in the "/Users/yourID/Music/iTunes/Mobile Applications" folder, the "/Users/yourID/Music/iTunes/iTunes Media/Mobile Applications" folder, or the "/Users/yourID/Music/iTunes/iTunes Music/Mobile Applications" folder.
    • Otherwise, put a game CD or DVD in the drive and choose it.
  • Then, it will ask you where to save the extracted files.
  • Next, you can choose whether to extract sounds, movies, or pictures.
  • Click the Extract Files button. After a lot of disk activity, you should have many ".wav", ".aif", ".mp2", ".mp3", ".caf", ".ogg", ".mov", ".m4v", ".avi", ".bik", ".pict", ".bmp", ".png", ".jpeg", ".tga", ".icns", ".ico", ".cur", ".txt", and/or ".html" files.

Windows version:

  • Install Java if you do not already have it.
  • Make sure that "Riveal.jar" is downloaded intact and not split apart. WinRAR seems especially keen to expand ".jar" files.
  • Check that Windows knows that ".jar" files should be opened with the Java Runtime Environment. The Windows Explorer should describe "Riveal.jar" as an Executable Jar File. If not, shift-right-click on "Riveal.jar" and choose the Open With... command. Pick "Java(TM) Platform SE binary" from the list of programs. (You may need to use the Choose Program... command and the Browse... button to find "javaw.exe". It will probably be in "c:\Program Files\Java\jre6\bin\" or something close to that.) Be sure to check "Always use the selected program to open this kind of file", then press OK.
  • If all goes well, double-clicking "Riveal.jar" will start Riveal.
  • It will ask you to point to a game's data files.
    • For most games, choose a folder (or a folder of folders) containing the game's application and data files. Many games will be in the "c:\Program Files" folder or a subfolder such as "Ubisoft" or "2K Games". Steam games are in the "c:\Program Files\Steam\steamapps\common" folder.
    • The GameTap version of Myst Online: Uru Live is in either "c:\Program Files\GameTap\games\140000150" or "c:\ProgramData\GameTap\games\140000150", depending on which version of Windows you have.
    • For an iOS game, you must choose the app file (".ipa") on your PC. First, find the iTunes data folder. It is either "c:\Documents and Settings\yourID\My Documents\My Music\iTunes" or "c:\Users\yourID\Music\iTunes", depending on your version of Windows. From there, look in the "Mobile Applications" subfolder, the "iTunes Media\Mobile Applications" subfolder, or the "iTunes Music\Mobile Applications" subfolder.
    • Otherwise, put a game CD or DVD in the drive and choose "d:\" or whatever letter corresponds to your drive.
  • Then, it will ask you where to save the extracted files.
  • Next, you can choose whether to extract sounds, movies, or pictures.
  • Click the Extract Files button. After a lot of disk activity, you should have many ".wav", ".aif", ".mp2", ".mp3", ".caf", ".amr", ".ogg", ".mov", ".m4v", ".avi", ".bik", ".pict", ".bmp", ".png", ".jpeg", ".tga", ".ico", ".cur", ".txt", and/or ".html" files.

Using the Extracted Files

To listen to, watch, and look at some of the sounds, movies, and pictures extracted by Riveal, you may need to install some additional software. There are many music players, movie players, and picture viewers available, so you may want to try a few until you find ones you like. Here are some that are known to work.

Any audio application should be able to play WAVE (".wav"), AIFF (".aif"), and MP3 files, including iTunes. iTunes can also play files in the MP2 format (i.e., layer 2 of MPEG-1 and MPEG-2), a precursor to MP3.

QuickTime Player can play CAF and AMR sound files.

To hear Ogg Vorbis (".ogg") sounds, you will need to use a music player that understands that format. If you install the Xiph QuickTime Components, all applications that use QuickTime (iTunes, for instance) should be able to play these sounds.

If you want to listen to the music extracted by Riveal on an iPod, you will need to use iTunes to convert MP2 and Ogg Vorbis files to AAC or MP3. You might want to also convert the WAVE and AIFF files to reduce their size.

You will need QuickTime Player to play the movies in MOV and M4V format. On Mac OS X, if you install Perian, QuickTime Player will be able to play many AVI movies. If you also install Flip4Mac WMV, QuickTime Player will also be able to play many WMV movies. On Windows, try Windows Media Player.

To play the Bink movies, you will need a special-purpose player. RAD Game Tools has players for Mac OS X and Windows at http://www.radgametools.com/bnkdown.htm.

All web browsers can display BMP, PNG, and JPEG pictures. QuickTime Player can also display these pictures and those in PICT and TGA formats, too.

Extracting from realMyst

Riveal can extract realMyst's sounds and movies from the data files on your hard disk or on CD. The sounds are all in WAVE format. The movies are mostly in QuickTime MOV format and some are in Bink (".bik") format. It can also extract the textures used to "paint" the surfaces of the objects in the game. The extracted textures are all PNG files. If you do not have a full installation of realMyst on your hard disk, you will want to extract files from your realMyst CD so that you get everything.

Riveal can also extract the images that are displayed on the Rime crystal viewer, which are encrypted in the "huevo.avi" and "vika.avi" movies. The decrypted pictures are BMP files in the "scn/maps/huevo" and "scn/maps/vika" subfolders.

The extracted sound files are placed in folders that correspond to the original realMyst ".dni" data files. Most of the music and the speeches are in the "mara" folder but a few are in the "pera" folder. Both folders also contain sound effects.

Extracting from BioShock Infinite

Riveal can extract sounds, movies, and textures from the Mac and Windows versions of BioShock Infinite and from the DLC (downloadable content) of the game. Extracted sounds are Ogg Vorbis (".ogg") and WAVE files, movies are Bink files, and textures are in PNG format.

There are many redundant copies of sounds and textures in BioShock Infinite and its DLC, so Riveal filters out the duplicates. To do so, it first processes the media in the main game's data files, then it processes each of the DLC folders in the order they were released. Only the unique sounds and textures of each new folder are extracted. As a result, people with different subsets of DLC are likely to see different extracted files in the DLC folders they have in common.
